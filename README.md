# Material básico sobre estatística
O material foi feito para aqueles que desejam dar uma refrescada na memória em alguns assuntos relacionados a estatística

### Neste material você vai ver
- Controle estatístico de processos
- Gráfico de Pareto
- Análise de componentes principais
- Gráfico de linha
- Diagrama de Ishikawa ou Diagrama de causa e efeito
